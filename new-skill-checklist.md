# New Skill Checklist

- [ ] Clarify task, scope, and audience.
- [ ] Decide personal vs project skill.
- [ ] Define inputs and outputs.
- [ ] Identify when Copilot should use the skill.
- [ ] Draft frontmatter (name, description, version, author).
- [ ] Draft body with purpose, workflow, examples.
- [ ] Ask user for review and refine.
- [ ] Suggest templates/scripts if helpful.
